# Game Maze

Current not playable, only generating maze with [backtracking method with stack](https://en.wikipedia.org/wiki/Maze_generation_algorithm#Iterative_implementation_(with_stack))

## Features

- Maze Generation Algorithm
- Maze Solver (not implemented)
- Playable (not implemented)

## Run

```bash
uv pip install -e .
uv run -m maze
```
